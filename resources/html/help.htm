Алгоритм Ахо-Корасик

Теоретические сведения
 
Скриншот окна
 

Использование программы:
  • При запуске программы автоматически создается бор с единственной корневой вершиной. Корневая вершина помечена красным цветом.
  • Для добавления строки в бор можно нажать "Add string" и добавить одну строку или добавить все строки из файла с помощью "Open strings".
  • Установить текст можно с помощью "Open text" или путём ввода текста в поле "Source text".
  • Кнопка "Step" делает один шаг алгоритма. Текущее положение помечается зелёным, полученные суффиксные ссылки - зелёной кривой, сжатые суффиксные ссылки - красной кривой.
  • Кнопка "Finish" делает шаги, пока выполнение не закончится.
  • С помощью "Undo" и "Redo" можно вернуть автомат в предыдущее состояние или в следующее, если таковое существует.
  • Кнопка "Restart" сбрасывает выполнение алгоритма, кнопка "Clear" очищает автомат.
  • Для сохранения состояния автомата можно использовать "File->Save graph".
